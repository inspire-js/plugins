# Overview

A thumbnail grid of your slides, to jump around quickly or get a bird's-eye view of the deck.

## Usage

- <kbd>Ctrl</kbd> + <kbd>H</kbd> (or <kbd>Alt</kbd> + <kbd>H</kbd>): section overview — section headers and the current slide.
- <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd>: overview of all slides.

You'll be prompted for a CSS selector to filter which slides are shown (leave empty for all). Click any thumbnail to go to that slide. Press <kbd>Esc</kbd> to exit.

## Autoload

Always loads (selector `*`), so the keyboard shortcuts are available in every deck.
